# Full Samwise Command List

### Placeholder Legend
|  Placeholder  |   Description         |
|   :---:       |    :---:              |
|   `[A]`       |   Any word or phrase  |
|   `[W]`       |   Any accepted key    |
|   `[N]`       |   Any number          |
|   `[T]`       |   Any unit of time    |
|   `...`       |   1 or more argument  |


## OPEN
* **Description**: Opens an App or website based on the arguments
* **Arguments**: `["[W]", "..."]`
    * `[W]` Options:   
      | "[W]" Value         | What it opens           |      Implemented?      |      
      | :---:               |  :---:                  |        :---:           | 
      | `WORD`              |   Microsoft Word        |           Yes          |
      | `POWERPOINT`        |   Microsoft Powerpoint  |           Yes          | 
      | `EXCEL`             |   Microsoft Excel       |           Yes          |
      | `STEAM`             |   Steam launcher        |           Yes          |
      | `DISCORD`           |   Discord app           |           Yes          |
      | `Canvas`            |   Canvas in browser     |           No           |
      | `YOUTUBE`           |   YouTube in browser    |           No           |
      | `DESTINY`           |   Destiny 2             |           No           |
      | `Siege`             |   R6 Siege              |           No           |
      | `[A].COM`           |   Opens in browser      |           No           |
      | `[A].NET`           |   Opens in browser      |           No           |
      | `[A].EDU `          |   Opens in browser      |           No           |
